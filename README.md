# Horse Racing Game: Introduction to Algorithms Mini-Project

This project simulates a horse racing game while demonstrating the use of fundamental algorithms and data structures — including Sorting, Searching, Insertion, and Deletion.

[Presentation Slide](https://www.canva.com/design/DAGylwh3mmE/PrnsqyaaN23la6WaOKvSBQ/view?utm_content=DAGylwh3mmE&utm_campaign=designshare&utm_medium=link2&utm_source=uniquelinks&utlId=hd1b3269a24)

**🧠 Algorithms Used**

*Sorting:* **Merge Sort** — used to rank horses on the leaderboard by total wins.

*Searching:* **Linear Search** — used to find a specific horse by name.

*Insertion:* Adds new horses into the race lineup (via list or linked structure).

*Deletion:* Removes horses from the lineup.

*Iteration:* Used to display all horses and display race progress.

*Randomized:* Mimicks the unpredictability of a horse race.

**🎯 Project Features**

- Race simulation with random movement.

- Final Stretch Tie-breakers.

- Top 5 Leaderboard sorted by wins.

- Add, remove, search, or display horses dynamically.
